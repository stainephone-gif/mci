# Flappy Bird - macOS Version

Нативное macOS приложение игры Flappy Bird, написанное на Swift с использованием SpriteKit.

## Описание

Это нативная реализация классической игры Flappy Bird для macOS. Управляйте птицей, кликая мышью или нажимая пробел, чтобы пролететь между зелеными трубами!

## Особенности

- Нативная реализация на Swift 5.0
- Использование SpriteKit для плавной анимации
- Поддержка управления мышью и клавиатурой (пробел)
- Сохранение лучшего результата
- Простая и интуитивная механика игры
- Минимальная версия: macOS 13.0+

## Требования

- macOS 13.0 или новее
- Xcode 14.0 или новее
- Swift 5.0

## Как собрать и запустить

### Вариант 1: Через Xcode

1. **Откройте проект в Xcode:**
   ```bash
   open FlappyBirdMac.xcodeproj
   ```

2. **Выберите схему:**
   - В верхней панели Xcode выберите схему `FlappyBirdMac`
   - Выберите целевое устройство: `My Mac`

3. **Запустите проект:**
   - Нажмите `⌘R` или кнопку ▶️ `Run`
   - Игра запустится в отдельном окне

### Вариант 2: Сборка через командную строку

```bash
# Перейдите в директорию проекта
cd FlappyBirdMac

# Соберите проект
xcodebuild -project FlappyBirdMac.xcodeproj -scheme FlappyBirdMac -configuration Release

# Приложение будет создано в:
# build/Release/FlappyBirdMac.app
```

### Вариант 3: Создание .app для распространения

```bash
# Соберите Release версию
xcodebuild -project FlappyBirdMac.xcodeproj \
    -scheme FlappyBirdMac \
    -configuration Release \
    -derivedDataPath ./build

# Готовое приложение:
# build/Build/Products/Release/FlappyBirdMac.app
```

## Управление

- **Клик мышью** или **Пробел** - птица подпрыгивает вверх
- Начало игры - клик или пробел на стартовом экране
- Перезапуск - клик или пробел после Game Over

## Архитектура

### AppDelegate
Управляет жизненным циклом приложения и создает главное окно.

### GameViewController
Настраивает SpriteKit view и представляет игровую сцену.

### GameScene
Основная игровая логика:
- Управление состоянием игры
- Обновление позиций объектов
- Обработка коллизий
- Подсчет очков
- Отрисовка UI

### Bird
Класс птицы с физикой:
- Гравитация
- Прыжки
- Обнаружение коллизий

### Pipe
Класс трубы:
- Генерация случайных позиций
- Движение слева направо
- Обнаружение столкновений

## Технологии

- **Swift 5.0** - язык программирования
- **SpriteKit** - 2D игровой фреймворк от Apple
- **AppKit** - фреймворк для macOS приложений

## Сборка для распространения

Для создания подписанного приложения:

1. В Xcode откройте проект
2. Настройте подпись: `Signing & Capabilities`
3. Выберите свою команду разработчика
4. Соберите Release версию
5. Архивируйте: `Product` → `Archive`
6. Экспортируйте приложение

## Решение проблем

### Приложение не запускается
- Проверьте версию macOS (требуется 13.0+)
- Убедитесь, что Xcode обновлен до версии 14.0+

### Ошибки сборки
- Очистите проект: `Product` → `Clean Build Folder` (⇧⌘K)
- Удалите DerivedData: `~/Library/Developer/Xcode/DerivedData`

### Проблемы с подписью
- Измените `CODE_SIGN_STYLE` на `Automatic` в настройках проекта
- Или отключите подпись для локальной разработки
